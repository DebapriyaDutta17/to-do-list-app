# 📝 To-Do List App

A modern and interactive To-Do List application built using **HTML, CSS, and JavaScript**. This project demonstrates core JavaScript concepts such as **DOM Manipulation**, **Event Handling**, and **Dynamic UI Updates**.

---

## 📌 Project Overview

The To-Do List App helps users organize daily tasks efficiently. Users can add new tasks, mark tasks as completed, and remove tasks when finished.

This project was developed as part of a JavaScript Mini Project to showcase front-end development skills and practical implementation of DOM manipulation.

## ⚙️ How It Works

### Adding a Task

1. Enter a task in the input field.
2. Click the "Add" button.
3. The task is dynamically added to the list.

### Completing a Task

1. Click on a task.
2. The task will be marked as completed.

### Deleting a Task

1. Click the Delete button.
2. The selected task will be removed instantly.
